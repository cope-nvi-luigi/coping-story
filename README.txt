# COPE-NVI Interactive Story - GitHub Pages Deployment

This package contains the necessary files to publish the COPE-NVI interactive story on GitHub Pages.

## Files Included
- index.html: The interactive story file (Twine-based).
- .nojekyll: Prevents GitHub Pages from processing the site with Jekyll.
- README.txt: This instruction file.

## Instructions to Publish
1. Create a GitHub account (if you don't have one).
2. Create a new public repository (e.g., coping-story).
3. Upload all the files from this ZIP into the repository.
4. Go to the repository settings > Pages.
5. Under "Source", select the `main` branch and root (`/`) folder.
6. Save and wait a few seconds.
7. Your site will be available at:
   https://<your-username>.github.io/<repository-name>/

Example:
https://cope-nvi-luigi.github.io/coping-story/
